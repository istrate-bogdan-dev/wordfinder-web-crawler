Terms of Use

WordFinder is a personal portfolio project by Bogdan Istrate. It is provided as an educational proof of concept for bounded website crawling, keyword discovery, and live crawl visualization.

Educational Portfolio Demo

This application is not a commercial crawling service, monitoring platform, legal compliance product, or security scanner. Results are intended for demonstration and learning purposes only.

Responsible Use

Use WordFinder only on websites that you own, manage, or are authorized to test. Do not use it to overload websites, bypass access controls, collect private data, or scan targets where automated access is not allowed.

Website Limits And Robots.txt

WordFinder includes practical limits such as crawl depth, page count, concurrency, response-size caps, same-domain crawling, and robots.txt checks. These controls reduce risk, but they do not make every scan appropriate. The user remains responsible for choosing safe targets and conservative settings.

Access Key

The live online demo may require an access key to prevent unwanted public scans. Local development usually does not require an access key unless one is configured by the operator.

Data And Logs

WordFinder processes the start URL, keyword, selected scan settings, and live scan results needed to run the crawler. The app does not provide user accounts and does not intentionally store scan result history. Hosting, reverse proxy, or infrastructure logs may still contain normal request metadata such as timestamps, paths, IP information, and user agent strings.

No Warranty

WordFinder is provided as-is. It may be unavailable, incomplete, produce inaccurate results, or stop a scan early because of network errors, robots.txt rules, content limits, or other safeguards. Use it at your own discretion.
